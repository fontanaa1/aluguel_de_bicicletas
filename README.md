Olá! Sejam bem-vindos ao projeto: API de Aluguel de Bicicletas.

Hoje, vamos construir uma API RESTful completa usando Node.js, Express e, o mais importante, o Supabase como nosso banco de dados.

O objetivo é criar uma API robusta para gerenciar o aluguel de bicicletas, implementando o CRUD e um filtro de disponibilidade. Vamos nessa!
